Jacob Dena Art 74 Portfolio The latest project I created is a sketch image made from processor. Processor is also the program I used to create my self portrait which took more time than this one. The picture shown here is a moving image where the image is created by ellipse circles with the colors of the image. The image is Master Chief from the Halo franchise series which is my favorite game growing up. When I discovered that I can do moving objects on processor I decided to do something like this. the circles is what creates the image in a moving effect which is what I enjoyed because its something different than a solid image that does not move.
